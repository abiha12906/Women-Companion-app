# Women Safety Companion App

A console-based Women Safety Companion App developed in **C++** using **Data Structures and Algorithms (DSA)**. The application helps improve personal safety through emergency assistance, safe route planning, cybercrime checking, and contact management.

---

## Features

- User Registration & Login
- Trusted Contact Management
- Safe Route Finder (Dijkstra's Algorithm)
- Nearby Safe Shelter Finder (BFS)
- One-Click SOS Emergency Alert
- AI-Based Safety Risk Analyzer
- Smart Travel Timer
- Cyber Crime Checker (Hash Table)
- Emergency Team Allocation (AVL Tree)
- Alert History (Stack)
- File Handling for Data Persistence
- Admin Panel

---
